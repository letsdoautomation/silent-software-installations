### Install command
```powershell
# With desktop shortcuts
cmd /c "GoogleDriveSetup.exe" --silent --desktop_shortcut --skip_launch_new
# Without desktop shortcuts
cmd /c "GoogleDriveSetup.exe" --silent --skip_launch_new --gsuite_shortcuts=false
```
